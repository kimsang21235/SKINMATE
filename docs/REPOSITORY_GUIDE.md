# Repository Guide

## Commit-ready checklist
- Remove local `instance/` before pushing
- Do not commit personal test images to `skinmate/static/uploads_temp/`
- Keep `.env` local only; share `.env.example`
- Run the app once after pulling to let runtime folders be recreated automatically

## Recommended GitHub repository name
`skinmate-ai-skin-analysis`

## Suggested topics
`flask`, `python`, `tensorflow-lite`, `sqlite`, `ai-service`, `portfolio-project`
